# Focus Timer

A ideia agora é criar uma aplicação chamada Focus Time, com objetivo de aplicar a tecnica de pomodoro.

Caracteristicas que aplicação tem

- Play, Pause, stop
- Som de fundo sendo possivel ativa ou desativa
